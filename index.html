New Power Method

My latest project is a new power method to obtain solutions of eigenvalue problems. The method can determine not only the dominant or lowest eigenvalues but also all eigenvalues without the need for a deflation procedure. The method uses a functional of an operator (or a matrix). The method can freely select a solution by varying a parameter associated to an estimate of the eigenvalue. Supporting files can be found on Github .
